CSS can be added to HTML documents in 3 ways:

  • Inline - by using the styel attribute inside HTML element
  • Internal - by using a style element in the head section
  • The most common way to add CSS., is to keep the style in external CSS files. However. in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

when comparing CSS class vs ID, the difference is that CSS class applies a style to one unique element. ID is also special in that you can use a special URL to an element and it's used by JavaScrept.

In CSS,selector are used to target a specific element or range of element on a web page. Once an element has been targeted,a style or set of style can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
